CountTableTool Documentation
============================

This document provides a minimal technical overview of the command-line interface for `CountTableTool`.

The tool is invoked via the main entry point in `CountTableTool.py` and exposes the following **subcommands**:

- `imputate`
- `col_ops`
- `export`
- `normalization`

Each subcommand has its own dedicated page:

- [Imputate](imputate.md)
- [Column operations (`col_ops`)](col_ops.md)
- [Export](export.md)
- [Normalization](normalization.md)

Refer to those pages for detailed descriptions of options and usage examples.
